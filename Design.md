# Design.md

## UI/UX улучшения, которые внесены
- Обновлена визуальная полоска главного меню: тёмно‑синий фон, белый текст, hover‑цвет текста красный.
- Выровнено меню по центру, убраны тени/transition, отключены focus‑ring эффекты.
- Установлена фиксированная высота полосы меню, чтобы она не “скакала” при клике.
- Устранено горизонтальное переполнение (убраны marginRight, расстояние между пунктами задаётся через gap).
- Унифицированы стили для десктопа и мобильного меню.
- Переименование бренда в интерфейсе на RenHub.
- Исправлена орфография “объявления”.
- Уменьшен заголовок “Публичные объявления” на 2 пункта (`text-2xl` → `text-xl`).

## Рекомендации по улучшению главной страницы
- Усилить CTA: “Добавить объявление” сделать контрастной кнопкой и добавить вторичную “Смотреть каталог”.
- Сократить заголовок Hero и увеличить информативность подзаголовка (1–2 ключевых пункта пользы).
- Карточки объявлений: добавить метаданные (город/район/цена/площадь/тип жилья).
- Укоротить описания до 2–3 строк и добавить “Подробнее”.
- Добавить счётчик объявлений в секциях “Новые/Популярные” и ссылку “Смотреть все”.

## Рекомендации по улучшению каталога
- Сжать вертикальные отступы между шапкой и контентом.
- Визуально выделить активную сортировку.
- Усилить поиск (контрастный фон, явный placeholder).
- Сгруппировать иконки действий, добавить tooltip/подписи.
- Добавить фильтры по цене, району, типу жилья.

## Рекомендации по улучшению “Мои объявления”
- Оставить один главный заголовок (H1) и снизить визуальный вес второстепенного.
- Сделать кнопку “+ Новое объявление” более заметной и ближе к заголовку.
- Карточки: заголовок жирный, описание 1–2 строки, метаданные отдельной строкой.
- Упростить визуальный шум (одинаковые размеры иконок, единые отступы).
- Добавить метку статуса “Публичное/Приватное” и отдельный счётчик лайков.
